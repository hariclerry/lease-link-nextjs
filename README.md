# LeaseLink
A property rental app that connects properties to renters

## Getting Started

First, run the development server:

```bash
npm run dev
```

## Tools used
 - Reactjs
 - Nextjs
 - Tailwind
 - Next-auth
 - Google Auth
 - Google Maps API
 - MapBox-GL
 - Photo swipe
